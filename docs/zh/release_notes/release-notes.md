# 发布说明

本节包含 FlagGems 的版本发布信息。

## V1.0.0

- **新增功能**：
    - 基础工作流：支持单步、固定流程的代码生成，包括 GroundTruth → TritonKernel → 正确性测试 → 性能测试。
    - 用户注册：支持自助注册与申请，试用需经平台审核批准。
    - Web 界面：可通过 https://kernelgen.flagos.io/ 在线访问。
    - 核心特性：
        - 全自动工作流：自动完成 AI 算子集的生成、测试与优化。
        - 多后端支持：无缝兼容多种 AI 框架与芯片，支持自动适配与调试。
        - 开箱即用：基于浏览器的操作界面，无需安装配置，也无需先验经验。
        - 标准化验证：自动生成测试用例，确保算子功能正确性。
        - 深度生态集成：与 FlagGems 和 FlagTree 协同，加速算子库开发。
